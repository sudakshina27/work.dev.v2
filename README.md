## 🚀 Running Locally

No build step needed. Just clone and open.

```bash
git clone https://github.com/sudakshina27/work.dev.v2.git
cd work.dev.v2
# Mac
open index.html
# Windows
start index.html
# or just open the file in your browser manually
```

Or use the **Live Server** extension in VS Code for hot reload.
